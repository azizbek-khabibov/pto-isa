# 谓词与物化指令集

这组指令负责在向量侧生成广播值、复制值以及与谓词结果直接相关的向量物化行为。它们连接标量/谓词状态与后续向量有效载荷运算。

## 常见操作

- `pto.vbr`
- `pto.vdup`

## 机制

- `vbr`：把标量或单值源广播成完整向量寄存器
- `vdup`：复制已有 lane 模式或单值到完整寄存器

这些操作常作为后续 `vsel`、`vcmps`、`vadd` 等向量操作的准备步骤。

## 约束

- 生成的向量宽度与目标元素类型一致
- 广播源的类型必须与目标寄存器类型兼容
- 不应把这些操作误写成普通内存加载
