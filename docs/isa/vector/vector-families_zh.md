# 向量指令族

向量指令族按功能和共享约束组织 `pto.v*` 微指令。每个族页先说明共同的机制、操作数模型、约束和 profile 缩窄，再进入具体 per-op 页面。

## 指令族概览

| 指令族 | 说明 |
| --- | --- |
| 向量加载存储 | UB 与向量寄存器之间的数据搬运 |
| 谓词与物化 | 广播、复制和向量侧谓词物化 |
| 一元向量操作 | 单输入 lane 级运算 |
| 二元向量操作 | 双输入 lane 级运算 |
| 向量-标量操作 | 向量与标量混合 |
| 转换操作 | 类型转换和索引生成 |
| 归约操作 | 跨 lane 或按 VLane 的归约 |
| 比较与选择 | 谓词生成和条件选择 |
| 数据重排 | interleave、pack、permute、slide |
| SFU 与 DSA | 特殊函数和领域专用操作 |

## 共享约束

- 向量宽度由元素类型确定
- 谓词宽度必须与向量宽度一致
- UB 指针和对齐要求必须与具体族匹配
- profile 缩窄必须单独说明，不能写成 PTO 默认语义
